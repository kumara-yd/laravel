## How to install

1. Open terminal
2. Run 

```shell
$ sh laravel.sh
$ Masukkan nama aplikasi Laravel: <input project name>
$ DB HOST: <input db host>
$ DB PORT: <input db port>
$ DB DATABASE: <input db database>
$ DB USERNAME: <input db username>
$ DB PASSWORD: <input db password>
```
